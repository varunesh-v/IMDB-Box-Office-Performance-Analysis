# IMDB-Box-Office-Performance-Analysis

A cloud-based business intelligence dashboard built using AWS QuickSight to analyze revenue trends, director performance, movie ratings, certifications, and genre-wise box office collections across the top 1000 IMDb films.

## 📌 Project Overview

This project provides interactive visual insights into movie industry performance using IMDb Top 1000 Movies data.

The dashboard helps identify:

- Highest grossing directors
- Revenue distribution by genre
- Top-rated movies
- Film certification trends
- Historical box office growth

## 🛠️ Tech Stack

- AWS QuickSight
- Amazon S3
- SPICE Engine
- CSV Dataset
- Data Visualization

## 📊 Dashboard Insights

### Top Grossing Directors
- Steven Spielberg generated the highest cumulative box office revenue.
- Anthony Russo and Christopher Nolan follow closely.

### Revenue by Genre
- Action & Adventure genres dominate worldwide earnings.
- Animation and Family movies show strong commercial success.

### Movie Ratings
- The Shawshank Redemption holds the highest IMDb rating among analyzed films.

### Film Certification
- U-rated films account for the largest share of movies in the dataset.
